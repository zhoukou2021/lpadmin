# LPadmin 快速上手指南

本指南将帮助您在5分钟内快速体验LPadmin后台管理系统的核心功能。

## 🚀 快速安装

### 前提条件
确保您的系统已安装：
- PHP 8.1+
- Composer
- MySQL 8.0+
- Node.js & NPM



### 手动安装

```bash
# 1. 克隆项目
git clone https://gitee.com/xw54/lpadmin.git
cd lpadmin

# 2. 安装依赖
composer install
npm install

# 3. 环境配置
cp .env.example .env
php artisan key:generate

# 4. 配置数据库（编辑.env文件）
nano .env

# 5. 数据库迁移
php artisan migrate
php artisan db:seed --class=LPadminSeeder

# 6. 发布资源
php artisan vendor:publish --tag=lpadmin-config
php artisan vendor:publish --tag=lpadmin-assets
npm run build

# 7. 创建管理员
php artisan lpadmin:create-admin

# 8. 启动服务
php artisan serve
```

## 🚨 常见问题

### 1. 登录问题
**问题**: 无法登录系统
**解决**:
```bash
# 检查数据库连接
php artisan migrate:status

# 重新创建管理员
php artisan lpadmin:create-admin

# 清除缓存
php artisan cache:clear
```

### 2. 权限问题
**问题**: 提示权限不足
**解决**:
```
1. 检查用户角色分配
2. 确认角色权限配置
3. 重新登录系统
```

### 3. 文件上传问题
**问题**: 文件上传失败
**解决**:
```bash
# 检查存储目录权限
chmod -R 775 storage/
chmod -R 775 public/storage

# 创建存储链接
php artisan storage:link
```

### 4. 静态资源404
**问题**: CSS/JS文件无法加载
**解决**:
```bash
# 重新发布资源
php artisan vendor:publish --tag=lpadmin-assets --force

# 重新编译前端资源
npm run build
```
